# MyBlog

#### My personal blog, based on Django implementation

1. 首页展示
2. 文章分类，并关联文章
3. 文章标签，并关联文章
4. 文章评论，验证码提交
5. 管理后台使用xadmin集成
6. 实现restfullAPI接口
7. 缓存实现
